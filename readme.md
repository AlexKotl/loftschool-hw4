# LOFT SYSTEM #
Backend part built with **Node**, **Express**, **Mongo** database and **Sockets**.
Live demo: https://loftschool-hw4.herokuapp.com/

- REST API
- JWT based auth
- Chat between users (sockets)
- User photo upload
- Mongoose ODM

Run `npm i && npm start` to build project
